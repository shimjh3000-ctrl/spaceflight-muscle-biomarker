# Data Directory

Raw sequencing data are NOT included in this repository (file size + licensing).
This directory holds only:

- `data/processed/` — small derived tables (CSV) generated by the pipeline
- `data/raw/` — *empty*; users download here following `docs/DATA_SOURCES.md`

## Reproducing the data set

All inputs are publicly accessible. See [`../docs/DATA_SOURCES.md`](../docs/DATA_SOURCES.md)
for accession-by-accession download instructions covering the 14 GEO/OSDR
datasets used in the meta-analysis.
